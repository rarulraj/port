Powerless
Choice Based RPG(Unity 2D)

Embarking on my first RPG project, I applied my expertise in Unity, implementing a bespoke dialogue system and a sophisticated weather cycle. This endeavor highlights my proficiency in C# and comprehensive understanding of intricate game systems. A noteworthy aspect of this project is the creation of 400 distinct sprites and pixel art tiles, meticulously crafted using Aseprite and GIMP. This not only demonstrates my artistic prowess but also showcases my commitment to delivering a technically refined and visually compelling gaming experience.

UDrone
Wave Based Shooter(Unreal Engine)

Entered the wave-based shooter domain, employing Unreal Engine's Blueprinting System to architect a dynamic gaming experience. Developed AI systems and particle effects from scratch, showcasing proficiency in Blueprinting and visual design. Explored the intricacies of Unreal Engine's Physics systems, demonstrating a nuanced understanding of in-game physics manipulation. Additionally, displayed competence in Unreal's UI system, ensuring a seamless and user-friendly interface for an enhanced gaming experience.

Rainbow Runner
Endless Runner(Unreal Engine)

Crafted an endless runner game featuring dynamic power-ups dispersed throughout the levels. Developed using C++ in conjunction with Unreal's blueprinting system, the project underscores proficiency in C++ programming and a nuanced grasp of level design principles. The game's map dynamically generates varied obstacles as the player progresses, showcasing a technical understanding of procedural level generation. This endeavor highlights a focus on technical intricacies, merging programming expertise with level design acumen to create a seamlessly engaging gameplay experience.

Voxel Shooter
A Multiplayer Shooter(Unreal Engine)

Developed an online multiplayer shooter featuring voxel-based environments as maps. The game offers a diverse range of classes for players to choose from at the onset, with multiple game modes like Team Deathmatch and Capture the Flag. The project showcases a profound understanding of server architecture and is implemented using C++. In this technical pursuit, Remote Procedure Calls (RPCs) play a pivotal role in facilitating communication between the server and clients, ensuring seamless interaction and synchronization within the multiplayer framework. The utilization of voxel-based environments demonstrates a focus on efficient memory management and optimization for online multiplayer scenarios. This project is a testament to a comprehensive grasp of networking protocols, server-side logic, and C++ programming within the context of multiplayer game development.

Magnet Puzzles
A VR puzzle game(Unreal)

Leveraged Unreal Engine's VR capabilities to construct a virtual reality (VR) experience featuring a magnetized gun mechanic. The mechanism enables the manipulation of world objects, strategically incorporating them into puzzles and environmental interactions within the level. The implementation involves a synthesis of C++ programming and Blueprinting to harness Unreal Engine's VR and animation functionalities. In this technical pursuit, the fusion of C++ and Blueprinting is executed with precision to orchestrate seamless interactions between the VR environment and the magnetized gun. The incorporation of VR animation principles is evident in the dynamic and responsive nature of the magnetized object interactions, providing a realistic and immersive experience. Furthermore, the project underscores a comprehensive understanding of puzzle-level design principles and broader game mechanics. The VR environment serves as a canvas for intricate puzzles, emphasizing the technical acumen required to design and implement engaging challenges within the immersive confines of virtual reality.
